From what I understood this course is to help give you the skills on how to use CSS, HTML and JS, to build web applications without the help of frameworks or libraries. We should prioritize the time to meet with our teams, but we should look a way to work as one team, never judge and help one another through each other’s struggles and we can use tools like Slack (Or MS Teams), GitHub and Trello to start the communication and be where we need to be.
