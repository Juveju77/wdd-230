Ch15 Modern JavaScript, Modular JavaScript

This section explains how you can use modules in JS. Modules are files with code that has the same objective. We learned about classes that encapsulates code with specific tasks, but this time all the code goes into one file which are called modules. The way to share functions or code from one module to another file is by exporting the files that will be used, and then in the other file importing only the pieces of the module that we need.
